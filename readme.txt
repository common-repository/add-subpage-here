Adds simple widget and admin menu item for adding a subpage to the current page.

== Description ==

This plugin makes it easy for you to add a new subpage to a page you are currently viewing. The plugin adds the following:

1. a widget which, when enabled, displays a "add subpage" link in your widget areas of pages;
2. a "Subpage" item to the "Add New" menu on the admin menu bar.

This plugin is a component of the [MIT Educational Collaboration Space](http://ecs.mit.edu) project.

== Installation ==

1. Upload the `add-subpage` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Optionally activate the "add subpage" widget.
